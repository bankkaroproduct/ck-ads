the unlock

Match the card to the cart.

CashKaro already shows a "best for Flipkart" card next to Flipkart deals. We can do that at the user level: target each shopper with the card that pays back where they already spend.

Flipkart and Amazon alone are 690K shoppers — each big enough for its own dedicated campaign with the exact card CashKaro already promotes.

recommendation

Notify Flipkart & Amazon shoppers first.

Lead with retailer-matched targeting. It resolves the size-vs-intent trade-off — the volume of the spend cohort with the relevance of intent, using a mechanic CashKaro already proves on its own homepage.

  1. Wave 1 — Flipkart shoppers (410K) with the Axis/SBI Flipkart card. Biggest pool, clearest match, ₹558 Cr of proven spend behind the pitch.
  2. Wave 2 — Amazon shoppers (280K) with the SBI Cashback card. Second-largest, same logic.
  3. Hold the card-intent cohort (321K) as a parallel test — different message ("finish what you started"), highest intent, compare conversion head-to-head.
  4. Test before scale — run a 2–3K slice per wave, measure click → application → card-out, then widen only if the funnel holds.
